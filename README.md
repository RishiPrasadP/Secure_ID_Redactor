# What it does
Automatically detects and masks sensitive data like SSNs and names from scanned ID documents.

##  How to run
1. Clone repo
2. `pip install -r requirements.txt`
3. `python app.py`
4. Visit `http://localhost:5000` in your browser

## Tech Stack
- Flask
- Tesseract OCR
- OpenCV
- Regex / spaCy
